What is AURORA‑ZK?

AURORA‑ZK is a next‑generation zero‑knowledge compliance engine designed for high‑performance, deterministic verification on Solana. Built across seven advanced languages, Aurora unifies circuits, proving, verification, governance, and developer tooling into a single cryptographic pipeline.

Core Architecture

Mojo Circuits

GPU‑accelerated zk‑circuits with deterministic proof generation.

Zig Verifier

Ultra‑efficient Solana on‑chain verifier optimized for low CU usage.

Vale Prover

Memory‑safe, linear‑type‑checked off‑chain prover service.

Gleam Compliance Engine

Deterministic rule evaluation with audit‑friendly policy bundles.

V‑Language SDK

Developer‑friendly CLI + SDK for proof generation and submission.

Ballerina Gateway

REST + gRPC orchestration layer for prover and verifier services.

Bosque Governance

Formally verified governance logic with deterministic upgrade rules.

Repository Structure

  • aurora-circuits/ — Mojo circuits + GPU kernels
  • aurora-prover/ — Vale prover service
  • aurora-verifier/ — Zig Solana program
  • aurora-compliance/ — Gleam policy engine
  • aurora-sdk/ — V‑language SDK + CLI
  • aurora-gateway/ — Ballerina API layer
  • aurora-governance/ — Bosque governance logic

Quick Start

Installation

Install the SDK package via npm or yarn:

npm install @gitdigital-products/solana-kyc-sdk
# or
yarn add @gitdigital-products/solana-kyc-sdk

Basic Usage

Initialize the SDK and perform a basic check:

import { KYCClient } from '@gitdigital-products/solana-kyc-sdk';
import { Connection, clusterApiUrl } from '@solana/web3.js';

const connection = new Connection(clusterApiUrl('devnet'));
const client = new KYCClient(connection, YOUR_PROVIDER_CONFIG);

// Example: Check a user's wallet status
const walletAddress = '...';
const status = await client.getVerificationStatus(walletAddress);
console.log(`KYC Status: ${status}`);
